<h1 align="center"> Challeng Alura- Telecom </h1>

<em> Descripción del proyecto: </em>
Se analiza las cancelaciones del servicio de Telecom 

## :hammer: Partes del analisis 
- `Extracción y ransformación`: la base de datos de Telecom debe ser procesada para el analisis 
- `Analisis descriptivo`: Se identifica el comportamiento estadistico de las principales variables 
- `Analisis de variables`: Identifica las posibles variables que esten relacionadas a las cancelacion de servicios de Telecom
- `Conclusiones y recomendaciones `: Se emiten conclusiones y sugerencias de los resultados
